## 🐛 التشخيص والحل

### س: التطبيق لا يظهر؟
**ج:**
1. تأكد من تشغيل `npm run dev`
2. افتح المتصفح على `http://localhost:5173`
3. امسح الكاش (Ctrl+Shift+Delete)
4. حاول مرة أخرى

### س: بيانات Firebase لا تظهر؟
**ج:**
1. تحقق من إضافة البيانات في Firebase Console
2. تأكد من Security Rules صحيحة
3. تحقق من الاتصال بالإنترنت
4. تحقق من الـ API Keys في firebase.ts

### س: خطأ في التسجيل؟
**ج:**
1. تحقق من البريد الإلكتروني
2. كلمة المرور يجب أن تكون 6+ أحرف
3. تأكد من وجود البريد في Firebase Authentication

### س: الأداء بطيء؟
**ج:**
1. تحقق من سرعة الإنترنت
2. امسح الكاش والملفات المؤقتة
3. أعد تشغيل المتصفح

## 💻 التطوير والكود

### س: أين تكون دوال Firebase؟
**ج:** في `src/services/firebaseService.ts`

### س: كيف أضيف ميزة جديدة؟
**ج:**
1. أنشئ مكون جديد في `src/app/components/`
2. استخدم دوال Firebase من `firebaseService.ts`

**ج:** عدّل `src/data/sampleData.ts`

**ج:** نعم، لكن ستحتاج لتعديل `firebaseService.ts`

## 🚨 المشاكل الشائعة والحلول

### المشكلة: Firebase لا يتصل
```
الحل:
1. تحقق من firebaseConfig في firebase.ts
2. تأكد من API Keys صحيحة
3. تحقق من قواعد الأمان
```

### المشكلة: البيانات الزائدة بطيئة جداً
```
الحل:
1. أضف pagination (تقسيم النتائج)
2. استخدم indexes في Firestore
3. قلل عدد الاستعلامات
```

### المشكلة: عدم ظهور التحديثات
```
الحل:
1. امسح الكاش (npm cache clean --force)
2. أعد تثبيت المكتبات (rm -r node_modules && npm install)
3. أعد تشغيل السيرفر
```
