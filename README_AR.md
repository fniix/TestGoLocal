# 🚀 GoLocal - نظام النقل المحلي

## 📱 نظرة عامة

**GoLocal** هو تطبيق متطور لحجز الرحلات والتوصيل في البحرين، يوفر:

- 🚗 **خدمة الرحلات** - احجز سيارة في دقائق
- 🚚 **خدمة التوصيل** - وصّل طلبك بسهولة
- 👥 **إدارة متقدمة** - لوحة تحكم للأدمن والدراؤفرز
- 📊 **بيانات حقيقية** - جميع البيانات من Firebase

---

## ✨ الميزات الرئيسية

### 👤 للمستخدمين
✅ حجز رحلات بسهولة
✅ متابعة الرحلة المباشرة
✅ تقييم الدراؤفرز
✅ محفظة رقمية آمنة

### 🚗 للدراؤفرز
✅ قبول الطلبات والرفض
✅ متابعة الأرباح
✅ تقييمات المستخدمين
✅ إدارة العروض

### 👨‍💼 للأدمن
✅ إدارة المستخدمين
✅ مراقبة الدراؤفرز
✅ التقارير المالية
✅ معالجة الشكاوى

## 🛠️ التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm أو yarn
- حساب Firebase

### التثبيت
```bash
# استنساخ المشروع
git clone <repository-url>
cd FinalGolocal

# تثبيت المكتبات
npm install

# تشغيل الخادم الخاص بك
npm run dev
```

### الوصول
```
تطبيق الويب: http://localhost:5173
```

## 🔐 البيانات والأمان

### Firestore Database
- **Collection**: `users`
  - المستخدمين العاديين
  - الدراؤفرز (role = 'driver')
  - الأدمن (role = 'admin')

### Firebase Authentication
- تسجيل الدخول الآمن
- حفظ الجلسات
- التحقق من الهوية

### Security Rules
- حماية البيانات الشخصية
- السماح بالقراءة والكتابة المصرحة فقط
